Extension: Estratégia de Vacinação Pesquisa

Official URL: http://www.saude.gov.br/fhir/r4/StructureDefinition/BREstrategiaVacinacaoPesquisa-1.0 Version: 0.1.0
Standards status: Informative Maturity Level: 1 Computable Name: BREstrategiaVacinacaoPesquisa

Extensão para as informações relacionadas aos dados da pesquisa clínica registrada na ANVISA.

Context of Use

This extension may be used on the following element(s):

  • Element ID Immunization.protocolApplied

Usage info

Usos:

  • Usa este Extensão: Imunobiológico Administrado em Rotina
  • Exemplos para este Extensão: Bundle/example-RIA-R

You can also check for usages in the FHIR IG Statistics

This structure is derived from Extension

Resumo

Extensão complexa: Extensão para as informações relacionadas aos dados da pesquisa clínica registrada na ANVISA.

  • numeroProtocoloEstudoANVISA: string: Número do protocolo do estudo clínico na Anvisa.
  • numeroVersaoProtocoloEstudo: string: Número da versão do protocolo do estudo na Anvisa.
  • numeroRegistroVacinaAnvisa: string: Número do registro sanitário da vacina na Anvisa, seja este registro emergencial ou definitivo.

Maturidade: 1
